# Spotify-Dashboard-Using-Power-BI

This project showcases an interactive Spotify Analytics Dashboard built using Power BI.  
It provides insights into song popularity, artist trends, explicit content distribution, release patterns, and album types using Spotify‚Äôs dataset.

## üöÄ Overview
The goal of this project is to convert raw Spotify track data into clear, easy-to-understand visual insights.  
The Power BI dashboard helps users analyze popularity trends, top artists/songs, music patterns across months/years, and key behavioral metrics from the dataset.

---

## üìå Features
- Fully interactive Power BI dashboard  
- KPI cards for:
  - Total Songs  
  - Distinct Artists  
  - Average Popularity  
  - Average Duration  
- Visual analysis of:
  - Top Artists by popularity  
  - Top Songs by popularity  
  - Explicit vs Non-Explicit content  
  - Popularity trend across months  
  - Distinct songs per year  
  - Album types distribution  
  - Release year distribution  
  - Song duration patterns  
- Clean and organized report layout with 3 dashboard pages
